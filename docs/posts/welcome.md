## 为什么选择 VitePress？

VitePress 是一个现代化的静态站点生成器，它具有以下优势：

1. **快速** - 基于 Vite 构建，开发体验极佳
2. **简洁** - Markdown 优先，专注于内容创作
3. **强大** - 基于 Vue 3，可以轻松扩展功能
4. **美观** - 现代化的默认主题，支持深色模式
